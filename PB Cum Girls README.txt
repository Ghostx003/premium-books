Three files:

📋 SETUP INSTRUCTIONS

Basic setup (no photos):
• Download pb_girls.html
• Open it in Chrome (recommended) or any modern browser
• That's it. Browse, search, click girls, play trailers.

With gallery photos (recommended):
• Download both pb_girls.html AND pb_girls_images.js
• Place them in the same folder
• Open pb_girls.html in Chrome
• The image sidecar loads automatically in the background — you'll see the status indicator in the header turn green when it's connected
• If it doesn't auto-load (some browsers block local file access), click the 🖼️ Image Manager button in the header and use 📜 Connect pb_girls_images.js to manually load it

With full-res photo folders (optional):
• Download PB_Images.zip and extract it
• In the dashboard, click 🖼️ Image Manager → 📁 Connect PB_Images Folder
• Select the extracted PB_Images folder
• Chrome will ask permission to read the folder — click Allow
• Now each girl's modal shows her full-resolution photos from the folder

Playing trailers:
• In any girl's open profile, look for pink scene lines with a ▶ icon
• Click to open a popup video player
• Multiple clips per scene? You'll get a clip selector bar
• Close with ✕ or press Escape
• Trailers stream from premiumbukkake.com — you need internet and site access for these to play
